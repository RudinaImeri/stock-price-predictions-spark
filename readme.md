# Stock Price Prediction with PySpark

This project is a **Stock Price Prediction System** built using **PySpark**, **Machine Learning**, and **Streamlit UI**.  
It processes financial data, trains models, and provides predictions through an interactive interface.

---

## Features

- Data ingestion using APIs
- Feature engineering with PySpark
- Machine Learning model training (GBT / Spark ML)
- Feature importance visualization
- Real-time stock prediction
- Multi-page Streamlit UI:
  - Data Overview
  - Model Training
  - Feature Importance
  - Predictions

---

## Tech Stack

- Python 3.10
- PySpark
- Pandas
- Scikit-learn
- Streamlit
- Anaconda (environment management)
